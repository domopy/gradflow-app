# AGENTS.md

## 代码规范

- 遵循项目现有代码风格、目录结构、命名习惯和架构边界。
- 优先复用已有工具函数、类型、组件和测试辅助方法。
- 不做与当前任务无关的重构、格式化或依赖升级。
- 不修改生成文件、锁文件或构建产物，除非任务明确要求。
- 代码注释默认使用中文，并使用UTF-8编码保存。
- 注释解释意图、约束、风险和边界，不解释显而易见的语法。
- 修改业务逻辑后必须添加或更新相关测试。
- 提交前运行lint、typecheck、test和build检查。
- 不硬编码密钥、令牌、账号、内部地址或敏感配置。
- 不删除或回滚用户已有改动。
- 公共接口、配置、环境变量、命令或数据格式改变时同步更新文档。

## Shell

- 默认使用PowerShell 7（`pwsh`）。
- 处理中文输出和文件时默认使用UTF-8。
