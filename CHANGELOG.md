# 更新日志

本项目遵循[Keep a Changelog](https://keepachangelog.com/zh-CN/1.1.0/)格式，并计划采用语义化版本。

## [Unreleased]

### Planned

- 更完整的备份文件选择器与提醒规则。

## [1.0.1] - 2026-06-22

### Added

- Android EAS Preview APK与Production AAB构建配置；
- JSON备份文件导出、系统分享和文件选择恢复；
- 日期时间输入组件交互测试，以及提醒和系统日历服务测试；
- 系统日历事件ID迁移，支持重复点击更新、编辑同步和删除清理；
- Git标签触发的GitHub Release工作流。

### Fixed

- 未完成的手动日期时间不会再被系统选择器覆盖；
- 提醒失败会区分平台不支持、缺少时间、权限拒绝、时间已过和系统错误；
- 设置页版本号与应用版本保持一致。

### Changed

- Android版本号更新为`versionCode 2`，应用版本更新为`1.0.1`；
- CI仅验证Android和Web，不再执行iOS导出。

## [1.0.0] - 2026-06-22

### Added

- 导入页消息发送时间、识别确认页、新建事项页和编辑事项页统一接入系统日期与时间选择器；
- 日期和时间可分开选择，取消选择不会修改原值；
- 时间选择逻辑覆盖手动输入保留、空值和本地时区午夜边界测试。

### Changed

- 所有日期时间输入继续兼容`YYYY-MM-DD HH:mm`手动格式；
- 完善README中的功能、配置、数据流、开发、验证和发布说明；
- 应用版本更新为`1.0.0`。

## [0.5.0] - 2026-06-21

### Changed

- 图片识别从设备端OCR切换为EasyOCR云API；
- 移除`expo-text-extractor`及Google ML Kit/Apple Vision运行时依赖；
- 图片上传前明确说明EasyOCR与DeepSeek两阶段数据流；
- Expo Go现在可以调试图片选择、压缩、EasyOCR和DeepSeek完整导入链路。

### Added

- EasyOCR服务地址与Access Key设置；
- `X-Access-Key`鉴权、multipart上传和响应Schema校验；
- 无效密钥、额度不足、上传错误、服务不可用与超时提示。

## [0.4.0] - 2026-06-21

### Added

- 延期、改期、换地点和取消消息与已有事项关联；
- 事项变更前后快照及详情页变更历史；
- 变更后本地提醒取消或重新调度；
- 确认页、日历和详情页时间冲突提示；
- 日历事项类型与课程筛选；
- 不含API Key、通知标识和图片的JSON备份、分享与整库恢复；
- 数据库v3迁移和v0.4核心逻辑测试。

## [0.3.0] - 2026-06-21

### Added

- 相册多选与拍照导入，单次最多5张图片；
- Android/iOS设备端OCR与分图证据文本；
- 图片压缩、临时文件清理和可配置的本地保留策略；
- 来源压缩图查看，以及删除最后一个关联事项后的文件清理；
- 相册、相机权限说明与Development Build文档。
