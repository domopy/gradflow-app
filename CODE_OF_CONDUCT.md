# 贡献者行为准则

## 我们的承诺

我们致力于为所有参与者提供友好、安全、包容且无骚扰的社区环境，不因年龄、身体状况、民族、性别认同与表达、经验水平、国籍、外貌、种族、宗教或性取向而区别对待。

## 期望行为

- 使用友善、包容的语言；
- 尊重不同观点和经验；
- 接受建设性反馈；
- 关注对社区最有利的事情；
- 对其他参与者保持同理心。

## 不可接受的行为

- 性化语言、图像或不受欢迎的性关注；
- 挑衅、侮辱、人身攻击或政治攻击；
- 公开或私下骚扰；
- 未经允许发布他人的私人信息；
- 其他在专业环境中不适当的行为。

## 执行

发现违反准则的行为时，请通过维护者公开资料中提供的私密联系方式报告。维护者有权删除、编辑或拒绝不符合准则的评论、提交、代码、Issue和其他贡献，并可临时或永久禁止违规参与者。

## 适用范围

本准则适用于项目空间，以及个人代表项目或社区公开发言的场合。
